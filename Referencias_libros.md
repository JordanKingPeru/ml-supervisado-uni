# Bibliografía recomendada del curso

Por derechos de autor este repo no incluye los libros; aquí está la referencia completa y dónde
conseguir cada uno (varios tienen versión oficial gratuita).

## Núcleo del curso

1. **Müller, A. C. & Guido, S.** — *Introduction to Machine Learning with Python*. O'Reilly Media.
   → [oreilly.com](https://www.oreilly.com/library/view/introduction-to-machine/9781449369880/) ·
   código oficial gratuito: [github.com/amueller/introduction_to_ml_with_python](https://github.com/amueller/introduction_to_ml_with_python)
2. **Siddiqi, N.** — *Credit Risk Scorecards: Developing and Implementing Intelligent Credit Scoring*.
   Wiley. (El dominio del dataset `credit_scoring.csv` del curso.)
   → [wiley.com](https://www.wiley.com/en-us/Credit+Risk+Scorecards%3A+Developing+and+Implementing+Intelligent+Credit+Scoring-p-9780471754510)
3. **Ng, A.** — Lecture Notes de *Machine Learning* (CS229, Stanford).
   → gratuito oficial: [cs229.stanford.edu](https://cs229.stanford.edu/) ·
   también *Machine Learning Yearning*: [deeplearning.ai](https://www.deeplearning.ai/)

## Complementarios

4. **Julian, D.** — *Designing Machine Learning Systems with Python*. Packt.
   → [packtpub.com](https://www.packtpub.com/)
5. *Applied Data Science* (lecture notes, Columbia University — Langmore & Krasner).
   → gratuito oficial: buscar "Applied Data Science Columbia lecture notes" (PDF de los autores).
6. *Lecture Notes in Machine Learning* — apuntes de curso (consultar cita exacta con el docente).
7. *Inteligencia Artificial Avanzada* — material en español (consultar cita exacta con el docente).

## Lecturas citadas en las sesiones

- **Sculley, D. et al. (2015).** *Hidden Technical Debt in Machine Learning Systems*. NeurIPS.
  → [papers.nips.cc](https://papers.nips.cc/paper/2015/hash/86df7dcfd896fcaf2674f757a2463eba-Abstract.html)
- **Kaufman, S., Rosset, S. & Perlich, C. (2012).** *Leakage in Data Mining*. ACM TKDD.
- **Lundberg, S. & Lee, S.-I. (2017).** *A Unified Approach to Interpreting Model Predictions* (SHAP). NeurIPS.
- **Molnar, C. (2022).** *Interpretable Machine Learning*.
  → gratuito oficial: [christophm.github.io/interpretable-ml-book](https://christophm.github.io/interpretable-ml-book/)
